# Install

From a [release](https://github.com/frantjc/forge/releases).

Using `brew`:

```sh
brew install --cask frantjc/tap/forge
```

From source:

```sh
git clone https://github.com/frantjc/forge
cd forge
make install
```

In GitHub Actions:

```yml
- uses: frantjc/forge@v2
```

As a Dagger module:

```sh
dagger install github.com/frantjc/forge
```
